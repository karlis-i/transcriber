
# Music Transcriber & Trainer

Clone of [Transcribe!](https://www.seventhstring.com/xscribe/overview.html) - *software to help transcribe recorded music* by [Seventh String Software](https://www.seventhstring.com/index.html)  

Goal: create a free and open source cross platform tool for musicians to transcribe and/or learn pieces of recorded music.  

Used technologies:
- Electron.js
- Node.js
- Bootstrap
